    Перемещение одного окна

    Окно, созданное с помощью window.open можно перемещать:

    • относительно текущего положения: moveBy(x, y); // moveBy(50, -50)
    • относительно экрана: moveTo(x, y); // moveTo(50, -50)

    Окну, созданному с помощью window.open можно изменять размер:

    • относительно текущего положения: resizeBy(width,height); // resizeBy(50, -50)
    • относительно экрана: resizeTo(width,height); // resizeTo(50, -50)
